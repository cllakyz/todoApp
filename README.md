# Laravel-Vue Todo App

> Basic Todo App

## Usage

Rename ".env.example" to "env" and update the values/settings to your own.

## Install Composer & Laravel

```
compoer install
```
```
php artisan key:generate
```
```
php artisan migrate:fresh --seed
```

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in serve
php artisan serve
```
